Herramientas de Control


En el presente trabajo de investigación exploramos el impacto que tienen sobre el desarrollo de software las herramientas de control de versiones y la realidad virtual. Se provee el contexto de donde se originan y como han evolucionado tales herramientas y tecnologías.

Git

Es la herramienta desarrollada desde 2005 y revoluciono el control de versiones.

GitHub

Es un servicio basado en la nube que aloja un sistema de control de versiones.

Gitlab

Es una plataforma de gestión de repositorios de código que utiliza Git.

Realidad Virtual (RV)


¿Como se podría aplicar el lenguaje c#? 

La Realidad Virtual, es una tecnología que usa software para crear un entorno artificial.

Gafas de Realidad Virtual

Son dispositivos que tienen sus propias pantallas incluidas dentro del mismo. Se pueden utilizar con teléfonos inteligentes, computadores de escritorio, portátiles, o televisión.
Realidad Aumentada - RA

Consiste en la integración de contenidos gráficos sobre una vista del mundo real.
Realidad Virtual - VR

Nos permite crear entornos digitales, virtuales, generados de forma artificial, dando la posibilidad de experimentar una inmersión completa en un entorno y la interacción con los diferentes objetos y escenas que se van dando en el mismo
